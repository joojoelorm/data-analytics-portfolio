# Donor Analytics Dashboard

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Project Overview
This dashboard was developed to track donor contributions and campaign performance.

## Features
- Donation Analysis by Campaign
- Monthly Donation Trends
- Donor Contribution Tracking
- Interactive Filters

## Key Insights
- Identified top-performing campaigns.
- Monitored donation patterns across months.
- Improved visibility into donor activity.
