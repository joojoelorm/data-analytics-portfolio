# Sales & Profit Analytics Dashboard

## Tools Used
- Power BI
- Data Visualization
- KPI Reporting

## Project Overview
This dashboard was developed to analyze sales performance, profit trends, and regional sales distribution.

## Features
- Monthly Sales Analysis
- Profit Trend Analysis
- Regional Performance Comparison
- KPI Cards for Sales, Profit, and Profit Margin

## Key Insights
- Identified high-performing regions.
- Monitored monthly profit fluctuations.
- Tracked overall business performance through KPI metrics.
